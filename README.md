# My Manga Library

Static manga reader for GitHub Pages.

## Features
- Multiple series support
- Drag & drop folder upload
- Next/Prev page buttons
- Remembers last read page per series
- Fully static, works offline

## How to use
1. Upload the repo to GitHub.
2. Enable GitHub Pages on `main` branch, root folder.
3. Open the link, drag & drop manga folders, and enjoy reading.
